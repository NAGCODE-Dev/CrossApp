# shared-ui

Limite inicial do design system interno do Ryxen.

## Objetivo

Servir `athlete` e `coach` com primitives pequenas e previsíveis.

## Entra primeiro

- `tokens`
- `primitives`
- `forms`
- `feedback`
- `layout`

## Não entra agora

- blocos de landing
- componentes de marketing
- componentes de domínio
- seções editoriais

## Regra

Se o componente precisar trocar copy ou estrutura para cada produto, ele ainda não pertence a `shared-ui`.
