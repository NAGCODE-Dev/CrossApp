# shared-contracts

Espaço reservado para contratos compartilhados entre frontend e backend.

## Alvo inicial

- schemas de auth
- payloads de athlete overview
- payloads de coach dashboard
- billing entitlements
- leaderboards e benchmarks

## Regra

Contrato compartilhado entra aqui quando já for usado por mais de uma superfície
ou quando reduzir drift entre frontend e backend.
