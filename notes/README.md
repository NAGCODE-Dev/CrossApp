# Obsidian Vault

Abra a raiz do repositório no Obsidian como vault.

Se quiser uma visão mais enxuta, use `notes/` como área principal de navegação dentro da vault.

Objetivo desta vault:

- guardar contexto vivo do projeto sem poluir a documentação operacional
- conectar sessões, decisões, roadmap, QA e produto
- funcionar junto com `docs/AGENT_CONTEXT.md`, não substituir

Regra prática:

- `docs/AGENT_CONTEXT.md`: handoff técnico curto e imediato
- `notes/`: contexto linkado, histórico e mapa mental de produto/arquitetura
