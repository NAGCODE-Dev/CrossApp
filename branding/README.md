# CrossApp Branding Kit

Arquivos base da identidade visual do CrossApp.

## Arquivos

- `crossapp-icon.svg` — ícone principal do app
- `crossapp-logo-horizontal.svg` — logo horizontal principal
- `crossapp-logo-white.svg` — versão monocromática branca
- `crossapp-logo-black.svg` — versão monocromática escura

## Paleta

- Fundo principal: `#040C17`
- Azul claro: `#59D1FF`
- Azul forte: `#2F8FFF`
- Laranja claro: `#FFBE55`
- Laranja forte: `#FF8A2A`
- Texto claro: `#F7FBFF`
- Texto secundário: `#C8D6E5`

## Tipografia sugerida

- Manrope
- Inter
- Sora
- Montserrat

## Uso sugerido

- `crossapp-icon.svg` para app icon, splash, manifest e avatar
- `crossapp-logo-horizontal.svg` para navbar, landing page e README
- versões monocromáticas para fundos sólidos, impressão e usos simples

## Exportações recomendadas

- `favicon-32.png`
- `apple-touch-icon.png` (180x180)
- `icon-192.png`
- `icon-512.png`
- `og-image.png`
